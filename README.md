# Backend

Laravel API for the online shop system.

Suggested stack:
- Laravel
- MySQL
- Sanctum for authentication

Suggested modules:
- Public API
  - Categories
  - Products
  - Product details
  - Search
- Private API
  - Wishlist
  - Cart
  - Checkout / create order
  - Order history
  - Profile update
  - Change password
  - Reviews
- Admin panel endpoints
  - Categories CRUD
  - Products CRUD
  - Orders management
  - Users management
